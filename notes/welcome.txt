Welcome to your Note-Taking App!

This is a simple, Apple Notes-like application with audio transcription capabilities.

Features:
- Create and organize notes in folders
- Plain text editing with standard shortcuts (Ctrl+C, Ctrl+V, Ctrl+Z)
- Drag and drop text files to create notes or insert content
- Drag and drop audio files for instant transcription using Whisper
- nested folder organization

Get started by:
1. Creating a new folder using the "New Folder" button
2. Creating a new note with "New Note"
3. Try dragging an audio file onto the editor to see transcription in action!

Enjoy taking notes!
